## Out of Scope (confirmed v1)

| Feature | Motivo |
|---------|--------|
| Login OAuth para clientes | Link + senha é suficiente e menos fricção |
| Integração com APIs de redes sociais | Meta API exige app review e infraestrutura complexa |
| App mobile nativo | Web-first; mobile via browser responsivo |
| Multi-stage workflows | Feature de enterprise, desnecessária para 10-30 clientes |
| White-labeling / domínio personalizado | Complexidade sem benefício imediato |
| IA para geração de legendas | Fora do escopo de aprovação |
| Drag-and-drop de artes | Conforto de UX para v2+ |
| Real-time collaboration | WebSockets desnecessário neste volume |
