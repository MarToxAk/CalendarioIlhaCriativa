## Phase Goal

**As a** cliente, **I want to** aprovar artes ou pedir alteração com histórico completo, **so that** eu e o admin possamos iterar até a arte estar correta.

<objective>
Preparar a fundação de dados e rotas para o fluxo de aprovação: migração que remove o índice único em approval_responses.arte_id (permitindo histórico), mudança de has_one para has_many no modelo Arte, extensão do validator para aceitar revised?, correção de check_deletable no admin controller e consolidação das rotas admin com a nova action mark_revised + rota nested de responses no portal do cliente.

Output: Migração executada, models atualizados, rotas consolidadas, suite completa verde.
</objective>
