| Boundary | Description |

| Model callback → ActionCable broadcast | broadcasts_revised_to_all roda server-side após commit; sem input externo |
| Broadcast → cliente específico | stream_for current_client garante que cliente X só recebe streams da própria arte |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-05 | Denial of Service | broadcasts_revised_to_all sem guard de admin | mitigate | `return unless admin` na primeira linha do método — sem crash se não houver User no banco |
| T-19-06 | Information Disclosure | render_partial_html em context de model | accept | Partials renderizados server-side; sem dados de outros clientes — client e arte sempre scoped ao registro atual |
| T-19-07 | Tampering | ActionView::RecordIdentifier.dom_id(self, "calendar_chip") | accept | ID gerado server-side a partir do ID do registro — cliente não controla o target |
| T-19-08 | Spoofing | stream_for current_client no canal | mitigate | stream_for cria stream scoped por record (client) — cliente X literalmente não recebe broadcasts de cliente Y mesmo se tentar subscrever |
| T-19-SC | Tampering | npm/pip/cargo installs | accept | Nenhum pacote instalado nesta fase — Package Legitimacy Gate não aplicável |
