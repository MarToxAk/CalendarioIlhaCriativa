# Wave 3 — View: Página de Configurações

## Context
Cria a view `admin/settings/show.html.erb` seguindo o padrão visual do projeto:
- Cards brancos com `rounded-xl border border-gray-200 shadow-card p-6 max-w-2xl`
- Inputs com `h-11 px-3 border border-gray-200 rounded-lg text-sm`
- Botões primários `h-9 px-4 bg-[#0F7949] hover:bg-[#0a5c37] text-white text-sm font-semibold rounded-lg`
- Labels `text-sm font-medium text-slate-900 mb-1`

## Tasks

### Task 1 — View show.html.erb

Criar `app/views/admin/settings/show.html.erb`:

```erb
<% content_for(:page_title) { "Configurações" } %>

<%# ===== CARD: Alterar Senha ===== %>
<div class="bg-white rounded-xl border border-gray-200 shadow-card p-6 max-w-2xl">
  <h2 class="text-sm font-semibold text-slate-900 border-b border-gray-100 pb-3 mb-5">
    Alterar senha
  </h2>

  <%= form_with url: update_password_admin_settings_path, method: :patch, class: "space-y-4" do |f| %>
    <div>
      <%= label_tag :password_current, "Senha atual", class: "block text-sm font-medium text-slate-900 mb-1" %>
      <%= password_field_tag :password_current, nil,
            autocomplete: "current-password",
            required: true,
            class: "block w-full h-11 px-3 border border-gray-200 rounded-lg text-sm text-slate-900 focus:outline-none focus:ring-2 focus:ring-[#0F7949]/20 focus:border-[#0F7949]" %>
    </div>

    <div>
      <%= label_tag :password, "Nova senha", class: "block text-sm font-medium text-slate-900 mb-1" %>
      <%= password_field_tag :password, nil,
            autocomplete: "new-password",
            required: true,
            class: "block w-full h-11 px-3 border border-gray-200 rounded-lg text-sm text-slate-900 focus:outline-none focus:ring-2 focus:ring-[#0F7949]/20 focus:border-[#0F7949]" %>
    </div>

    <div>
      <%= label_tag :password_confirmation, "Confirmar nova senha", class: "block text-sm font-medium text-slate-900 mb-1" %>
      <%= password_field_tag :password_confirmation, nil,
            autocomplete: "new-password",
            required: true,
            class: "block w-full h-11 px-3 border border-gray-200 rounded-lg text-sm text-slate-900 focus:outline-none focus:ring-2 focus:ring-[#0F7949]/20 focus:border-[#0F7949]" %>
    </div>

    <div class="pt-1">
      <%= f.submit "Alterar senha",
            class: "h-9 px-4 bg-[#0F7949] hover:bg-[#0a5c37] text-white text-sm font-semibold rounded-lg transition-colors cursor-pointer" %>
    </div>
  <% end %>
</div>

<%# ===== CARD: Dados da Agência ===== %>
<div class="bg-white rounded-xl border border-gray-200 shadow-card p-6 max-w-2xl mt-6">
  <h2 class="text-sm font-semibold text-slate-900 border-b border-gray-100 pb-3 mb-5">
    Dados da agência
  </h2>

  <%= form_with url: update_agency_admin_settings_path, method: :patch, class: "space-y-4" do |f| %>
    <div>
      <%= label_tag :agency_name, "Nome da agência", class: "block text-sm font-medium text-slate-900 mb-1" %>
      <%= text_field_tag :agency_name, Current.user&.agency_name,
            required: true,
            maxlength: 100,
            class: "block w-full h-11 px-3 border border-gray-200 rounded-lg text-sm text-slate-900 focus:outline-none focus:ring-2 focus:ring-[#0F7949]/20 focus:border-[#0F7949]" %>
      <p class="mt-1 text-xs text-slate-400">Aparece no topo do menu lateral.</p>
    </div>

    <div class="pt-1">
      <%= f.submit "Salvar",
            class: "h-9 px-4 bg-[#0F7949] hover:bg-[#0a5c37] text-white text-sm font-semibold rounded-lg transition-colors cursor-pointer" %>
    </div>
  <% end %>
</div>
```

**Notas:**
- Usamos `form_with url:` (não `model:`) porque não há objeto ActiveRecord sendo editado diretamente no form
- `password_field_tag` e `text_field_tag` em vez dos helpers de model para evitar binding a um objeto
- `Current.user&.agency_name` pré-preenche o campo com o valor atual
- Os dois formulários são independentes — erro num não afeta o outro
