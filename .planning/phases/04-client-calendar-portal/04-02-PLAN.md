<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → Client::HomeController#index | params[:month] é input do usuário (URL manipulável) |
| controller → database | Query de artes deve ser sempre escopada por @client |
| view → client_arte_path | IDs de arte na URL são input do usuário (cruzamento de clientes) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02-01 | Elevation of Privilege | Client::HomeController — query de artes | mitigate | `@client.artes.where(...)` — nunca `Arte.where(...)` direto; teste de isolamento em client_isolation_test.rb |
| T-04-02-02 | Tampering | params[:month] — Date.strptime | mitigate | `rescue Date::Error` → fallback para `Date.today.beginning_of_month`; valor inválido não causa 500 |
| T-04-02-03 | Spoofing | external_url em artes renderizadas na grade | accept | Grade só exibe ícone + badge (sem renderizar external_url); risco de open redirect só existe na página de preview (Plano 03) |
| T-04-02-04 | Denial of Service | N+1 queries na grade com muitas artes | mitigate | `.includes(media_file_attachment: :blob)` na query do controller — carregamento eager |
| T-04-02-SC | Tampering | npm/pip/cargo installs | accept | Nenhuma instalação de pacotes neste plano |
</threat_model>
